Department & Employee Management System (C++)

This C++ project demonstrates a Department and Employee management system using dynamic arrays. The project allows you to:

Hire employees dynamically (with automatic array resizing)

Fire employees

Search for employees by ID

Display department information along with currently employed staff

It is designed to help practice object-oriented programming, dynamic memory management, and basic data structure handling in C++.
